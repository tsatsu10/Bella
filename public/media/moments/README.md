# Moments folder

**Add images here** — they will appear automatically in the Moments gallery on the site.

- Supported formats: `.jpg`, `.jpeg`, `.png`, `.webp`, `.gif`
- Images are listed in alphabetical order by filename
- The **alt text** used for accessibility is the filename without the extension (e.g. `Beach day.jpg` → "Beach day")

After adding or removing files, refresh the page to see the updated gallery.
