# Playlist folder

**Add audio files here** — they will appear automatically in the music player playlist.

- Supported formats: `.mp3`, `.m4a`, `.ogg`, `.wav`, `.aac`
- The **track label** in the player is the filename without the extension (e.g. `Can I Call You Rose.mp3` → "Can I Call You Rose")
- Files are listed in alphabetical order

After adding or removing files, refresh the page (and restart the dev server if needed) to see the updated playlist.
